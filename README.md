# Lex-Yacc-Programs

## Installation
```
sudo apt-get install flex
sudo apt-get install byacc
sudo apt-get install bison
```
## Executing
```
git clone https://github/com/ranaaditya/Lex-Yacc-Programms.git
cd Lex-Yacc-Programms
lex <file_name>.l
gcc lex.yy.c
./a.out
```
